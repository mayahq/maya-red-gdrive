Description

This node searches for files and folders in your google drive based on an input query

Session

Select or Create a gdrive-auth config node

Inputs

querystring
search query combining one or more search terms
includeItemsFromAllDrivesbool
Whether both My Drive and shared drive items should be included in results.
pageSizenum
The maximum number of files to return per page. Partial or empty result pages are possible even before the end of the files list has been reached. Acceptable values are 1 to 1000, inclusive. (Default: 100)
pageToken string
The token for continuing a previous list request on the next page. This should be set to the value of 'nextPageToken' from the previous response.

Outputs

payloadvarious
Contains the list of files that match the query
