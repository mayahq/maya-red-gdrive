Description

Google Drive Export node is to download files from Google Drive in various available format

Config

Document Url string
URL of the Google drive document to be downloaded
File Type string
Select the type of file to be downloaded from the available options: Google Doc (doc), Google Sheet (sheet) or Google Slide (slide)
Export Type string
Choose the format of file to be downloaded as.
Save to string|path
Save the exported file to a location on disk. Leaving this blank will save the file in $HOME directory of device
Filename string
Save the file as this name on the filesystem. Do not put the file extension here

Outputs

payload file
The downloaded file is available as payload of the node
