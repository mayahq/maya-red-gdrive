Description

Google Sheet Append node is used to append data to an existing table in a google sheet document

Session

Select or Create a gdrive-auth config node

Inputs

url string
URL of google sheet document
range string
The Range of Table in the following format: sheet_name!start_cell:end_cell
values array
2D array of data that needs to be inserted into the sheet
majorDimension enum
ROWS: inserts data horizontally, COLS: inserts data vertically
valueInputOption enum
How the input data should be interpreted.
insertDataOption enum
How the input data should be inserted.
responseValueRenderOption enum
Determines how values in the response should be rendered.
responseDateTimeRenderOption enum
Determines how dates, times, and durations in the response should be rendered. This is ignored if responseValueRenderOption is FORMATTED_VALUE.

Outputs

payload various
Contains the http response from gsheet append values api
